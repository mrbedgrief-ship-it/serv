CRIPCRAFT - READ ME FIRST
=========================

This is a REAL local prototype for Windows.
It is NOT a finished commercial server.
For local testing, username verification is DISABLED on purpose.
That means you should be able to join your own PC without the "Failed to verify username" error.

STEP 1 - Install Java 21
- If you do not already have Java 21, install it first.
- Recommended download: Eclipse Temurin 21 (Windows x64 MSI).

STEP 2 - Download the Minecraft server file
- Double-click: scripts/setup.bat
- Wait for it to finish.
- This downloads the official Minecraft Java server jar into:
  downloads/vanilla-server-1.20.4.jar

STEP 3 - Open the launcher
- Double-click: launcher/start-launcher.bat

STEP 4 - Check files
- In the launcher, click: Check Files
- Confirm that it says local login mode is OK.

STEP 5 - Start CripCraft
- In the launcher, click: Play
- The first time, type YES when asked to accept the Minecraft server EULA.
- Wait until the server window says Done.

STEP 6 - Join in Minecraft Java Edition
- Open Minecraft Java Edition
- Use version: 1.20.4
- Multiplayer -> Add Server
- First try this address: localhost
- If localhost does not work, try: 127.0.0.1

STEP 7 - What you should see
- You should spawn in the CripCraft hub.
- Use the hub buttons to enter Zone Survival.

If something goes wrong, open BEGINNER_LOGIN_FIX.txt, STATUS.md, and README.md.
