CRIPCRAFT - READ ME FIRST
=========================

This is a REAL local playable Minecraft game prototype for Windows.
Turn shaders OFF before testing.

STEP 1
- Install Java 17.

STEP 2
- Double-click: scripts/setup.bat
- Wait for it to finish.
- If browser tabs open, download the missing mod files and place them exactly where the script says.

STEP 3
- Double-click: launcher/start-launcher.bat

STEP 4
- In the launcher, click: Check Files

STEP 5
- In the launcher, click: Play

STEP 6
- A server window will open.
- Then Prism Launcher will open the CripCraft game instance.
- If Prism asks you to sign into your Minecraft account the first time, do that there.

STEP 7
- If the server is still loading, wait until it says Done.
- The game should connect to localhost automatically.
- If it does not connect, use Multiplayer and join: localhost

STEP 8
- You should spawn inside Outpost Blackline.
- Press the Zone Survival button.
- Go outside the safe area, scavenge, survive, and come back alive.

If the game looks gray or broken:
- Turn shaders OFF
- Close the game
- Read README.md and STATUS.md
