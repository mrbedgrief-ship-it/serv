CRIPCRAFT - READ ME FIRST
=========================

This is a REAL local Forge prototype for Windows.
This build is for LOCAL TESTING.
Turn shaders OFF before testing.

STEP 1
- Install Java 17.

STEP 2
- Double-click: scripts/setup.bat
- Wait for the script to finish.
- If browser tabs open, download the missing mod files and place them exactly where the script says.

STEP 3
- Double-click: downloads/forge-1.20.1-47.4.10-installer.jar
- In the Forge installer window, click: Install client
- Wait until it finishes.

STEP 4
- Copy every file from:
  client-pack/mods/
  into your Minecraft mods folder:
  %APPDATA%/.minecraft/mods/

STEP 5
- Double-click: launcher/start-launcher.bat

STEP 6
- In the launcher, click: Check Files
- If everything looks OK, click: Play

STEP 7
- Wait until the server window says Done
- Open the normal Minecraft Launcher
- Pick the Forge 1.20.1 profile
- Make sure shaders are OFF
- Join this address first: localhost
- If localhost fails, try: 127.0.0.1

STEP 8
- You should spawn inside the CripCraft Hub bunker
- Press the Zone Survival button
- Explore the quarantine zone and survive

If the world looks gray or broken:
- Turn shaders OFF
- Close the game
- Read README.md and STATUS.md
